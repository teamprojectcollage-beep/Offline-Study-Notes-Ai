# Offline Study Notes AI Assistant

A simple offline AI tool that helps students summarize and organize their study notes using a local language model (Mistral).

## Features
- Works completely offline after one-time model download
- Summarizes long notes into bullet points and key sections
- Saves all notes and summaries locally
- Simple GUI built with Tkinter

## How to Run
1. Install Ollama from https://ollama.com
2. Run in Command Prompt: `ollama run mistral`
3. Install package: `pip install ollama`
4. Run the app: `python main.py`
